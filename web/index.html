NJ Transit Simulator

uses open data from NJTransit to display a simulation showing all of the currently active NJTransit trains and busses.

How?

A batch process imports NJTransit's GTFS data into a more easily queryable database. A server-side script runs continually to create a listing of current vehicle positions that your web browser is downloading whenever the loading indicator below is shown.
